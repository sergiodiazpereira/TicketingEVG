# Plan de Pruebas Unitarias y de Integración - TicketingEVG

Este documento contiene el plan de pruebas formal para el sistema **TicketingEVG**. Las pruebas cubren tanto validaciones de interfaz de usuario (Frontend Angular), integración con el inicio de sesión único (SSO Catch), control de permisos y ciclo de vida de los tickets, como persistencia y validación en la base de datos (Backend PHP).
